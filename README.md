# 🏥 Hospital Insights Dashboard (Power BI)

## 📌 Project Overview

This project presents an **interactive Power BI dashboard** designed to analyze hospital data and generate meaningful insights. It helps in understanding patient trends, hospital performance, and key healthcare metrics.

---

## 🎯 Objectives

* Analyze patient admissions and discharge trends
* Monitor hospital performance and efficiency
* Track key healthcare KPIs
* Provide data-driven insights for decision-making

---

## 📊 Key Features

* 📈 Interactive dashboards and visualizations
* 🧮 DAX-based calculations and measures
* 🏥 Patient admission & discharge analysis
* 💰 Revenue and cost insights (if applicable)
* 📍 Department-wise performance tracking

---

## 📂 Project Structure

```
Hospital-Insights-Dashboard/
│
├── Hospital Data Report.pbix      # Power BI report file
├── Hospital.xlsx                # Dataset used for analysis
├── README.md                # Project documentation
```
---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **Microsoft Excel / CSV** – Data source
* **DAX (Data Analysis Expressions)** – Calculations and measures

---

## 📊 Key KPIs Used

* Total Patients
* Admission Rate
* Discharge Rate
* Average Length of Stay
* Bed Occupancy Rate
* Department-wise Patient Count
